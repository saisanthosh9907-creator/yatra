# YatraCost Pro – Trip Cost Estimator

This repo contains a front-end only version of **YatraCost Pro**, a trip cost calculator that runs fully in the browser using HTML, CSS and JavaScript.

## How to host on GitHub Pages

1. Create a new public repository.
2. Upload this `index.html` file (and others if any).
3. In **Settings → Pages**, set:
   - Source: `main` branch
   - Folder: `/ (root)`
4. Save. Your site will be available at:

   `https://<your-username>.github.io/<repo-name>/`

## Features

- From / To / Via route entry
- Distance lookup via OSRM + OpenStreetMap
- GPS-based current location
- Weather at destination (Open-Meteo)
- Driving and flying cost modes
- Stay, food, extras and per-person cost calculation
- Dark / light theme toggle
- Shareable URL with query params
- LocalStorage persistence for state and user login
